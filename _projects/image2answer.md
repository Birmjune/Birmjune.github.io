Submission for the **Samsung Collegiate Programming Challenge: AI track**
(advanced to the finals).

**Image2Answer** is a vision-language pipeline that automatically solves
image-based four-option multiple-choice questions: it reads a question image and
predicts the correct choice among the four options.

- **Task:** Image-based 4-option MCQ.
- **Approach:** A vision-language pipeline (see the repo for details).
- **Result:** Advanced to the finals.

Code and details on [GitHub](https://github.com/Birmjune/Image2Answer-4optionMCQ).
