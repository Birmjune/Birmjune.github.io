# Birmjune Kim — personal website

Source for [Birmjune.github.io](https://birmjune.github.io), built with Jekyll and based on the [al-folio](https://github.com/alshedivat/al-folio) theme.

## Local development

```sh
bundle install
bundle exec jekyll serve
```

Site content lives in `_pages`, `_news`, `_projects`, and `_data/cv.yml`.
